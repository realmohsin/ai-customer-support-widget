Demo of AI Customer Support Widget

On this page, you can find a demo of the customer support widget that can be added to any website. From the widget, you can chat with an AI agent by text or voice, and it will escalate to a human when requested.

Visit the admin dashboard page to view conversations and configure the widget and AI agent. From the admin dashboard, you can respond to escalated conversations.

For this demo, the agent is configured to provide support for a bank called VapiBank. Try out the voice call feature if a microphone is available.

Enjoy the demo!
